main方法是第一个入栈的，因为它是程序最先执行的方法并且有权力去调用其它的方法，new出来的对象，数据，放在堆里面，堆会开辟出内存的空间并初始化值char类型初始化的0x0000，是因为一个char占两个字节，16位，全部初始化成了0.0x0011是地址的编号，根据地址编号找到数据。

栈内存里面存储的变量必须赋值（否则报错），堆里面的不是必须赋值，因为系统在划分内存空间的时候已经赋了初始值。
