###字符类演示

  • A:字符类
    • [abc] a、b 或 c（简单类）
    • [^abc] 任何字符，除了 a、b 或 c（否定）

数量词

  • A:Greedy 数量词
    • X? X，一次或一次也没有
    • X* X，零次或多次
    • X+ X，一次或多次
    • X{n} X，恰好 n 次
    • X{n,} X，至少 n 次
    • X{n,m} X，至少 n 次，但是不超过 m 次

  • A:正则表达式的分组功能

    • 捕获组可以通过从左到右计算其开括号来编号。例如，在表达式 ((A)(B(C))) 中，存在四个这样的组： 
  •  1     ((A)(B(C))) 
 2     (A 
 3     (B(C)) 
 4     (C) 

 组零始终代表整个表达式。
